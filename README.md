## Todos

Twitter clone built using Rails 5.2.

## Technologies
- The minimum Ruby version is 2.3
- RSpec - testing

## Usage
```
$ git clone git@github.com:alexscotttonge/Peepr
$ cd Peepr
$ bin/rails server
```

## Testing

Install ```rspec```

```gem install rspec```

Then run tests:
```$ rspec```

Test coverage: 99.5%
